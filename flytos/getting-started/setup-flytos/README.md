# Setup FlytOS

FlytOS is compatible with ODROID-XU4, Raspberry Pi 3B/3B+, NVIDIA Nano, NVIDIA Jetson TX1/TX2 , Intel Edison and Intel Aero Compute Board. We will soon introduce compatibility with many popular SBCs available in the drone ecosystem. Select your companion computer from the links below for tutorial to install FlytOS:

* [Odroid XU4](odroid-xu4/)
* [Raspberry Pi 3B/3B+](raspberry-pi-3b-3b+/)
* [NVIDIA Nano](nvidia-nano.md)
* [NVIDIA TX1/TX2](nvidia-tx1-tx2.md)
* [Intel Edison](intel-edison.md)
* [Intel Aero Compute Board](intel-aero-compute-board.md)
